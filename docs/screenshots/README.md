# Screenshots

Coloca aquí las capturas de pantalla referenciadas en el README principal.

## Capturas requeridas

| Archivo | Sección | Cómo capturar |
|---------|---------|----------------|
| `dashboard.png` | Dashboard principal | Login con `demo@dian.co` / `demo1234`, vista inicial |
| `invoices.png` | Página Facturas | Navega a `/invoices` con filtros aplicados |
| `reports.png` | Reportes PDF | Navega a `/reports`, selecciona Form 300, vista previa |
| `calendar.png` | Calendario DIAN | Navega a `/calendar` |
| `companies.png` | Multi-empresa | Navega a `/companies` con varias empresas |
| `palette.png` | Command Palette | Presiona `Cmd+K` desde cualquier página |

## Recomendaciones

- Resolución mínima: **1440 × 900** para desktop
- Formato: PNG con compresión OK (TinyPNG si pesa >500KB)
- Modo claro para mejor legibilidad en GitHub
- Captura solo el contenido, sin barra del navegador

## Generación rápida

```bash
npm run dev
# Abre http://localhost:5173
# Captura cada vista con Win+Shift+S (Windows) o Cmd+Shift+4 (Mac)
# Guarda con los nombres exactos arriba
```
